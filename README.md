## ✨ Возможности

- 🔄 Перевод чисел между системами счисления (2-36)
- 🧮 Арифметические операции в любой системе счисления
- 📊 Поддержка дробных чисел
- 🔧 Побитовые операции (AND, OR, XOR)
- 🎨 Красивый TUI интерфейс с цветами
- ⚡ Быстрая работа без GUI

## 🚀 Установка и запуск

```bash
# Сделать файл исполняемым
chmod +x tui_calculator.py

# Запустить приложение
python3 tui_calculator.py

# Или
./tui_calculator.py
```

## 🎮 Управление

- **↑↓** - навигация по меню
- **Enter** - выбор пункта меню
- **q** - выход из программы
- **Esc** - возврат в главное меню

## 📋 Требования

- Python 3.6+
- Терминал с поддержкой curses
- Linux/macOS (Windows может потребовать дополнительной настройки)

## 🔧 Поддерживаемые операции

- Сложение (+)
- Вычитание (-)
- Умножение (*)
- Деление (/)
- Возведение в степень (**)
- Остаток от деления (%)
- Побитовое AND (&)
- Побитовое OR (|)
- Побитовое XOR (^)

## 📝 Примеры использования

1. **Перевод чисел:**
   - Введите число: `FF`
   - Из системы: `16`
   - В систему: `10`
   - Результат: `255`

2. **Арифметические операции:**
   - Первое число: `1010`
   - Второе число: `1100`
   - Операция: `+`
   - Система счисления: `2`
   - Результат: `10110`

## 🐛 Устранение неполадок

Если возникают проблемы с отображением символов:
```bash
export LANG=en_US.UTF-8
export LC_ALL=en_US.UTF-8

```
